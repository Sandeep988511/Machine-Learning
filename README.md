# MLP Depth Tutorial

This project demonstrates how varying the depth of a Multilayer Perceptron affects learning behaviour using the `make_moons` dataset.

## Contents
- Jupyter Notebook
- Tutorial PDF
- Saved figures

## How to Run
1. Install requirements:
   pip install numpy matplotlib seaborn scikit-learn

2. Run the notebook:
   jupyter notebook MLP_depth_analysis.ipynb
